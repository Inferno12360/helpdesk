#### By Key
`getprioritaetbykey`
![[Priorität_01.png]]
Durch das Feld "key" (markiert durch Rot im Bild) sucht dieser Endpunkt nach einer Priorität welcher, diesen bestimmten PK hat.

```json title:"Beispiel Ausgabe"
[
  {
    "PK_Priorität": 1,
    "Prioritätsname": "Hoch",
    "Beschreibung": "Wichtige Aufgabe"
  }
]
```
