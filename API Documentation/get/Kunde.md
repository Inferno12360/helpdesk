#### By Key
`getkundebykey`
![[Kunde_01.png]]

Durch das Feld "key" (markiert durch Rot im Bild) sucht dieser Endpunkt nach einem Kunden welcher, diesen bestimmten PK hat.

```json title:"Beispiel Ausgabe"
[
	{
		"PK_Kunde": 1,
		"Kundennummer": "K001",
		"Firmenname": "Firma Alpha"
	}
]
```
