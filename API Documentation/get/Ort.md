#### bykey
`getortbykey`
![[Ort_01.png]]
Durch das Feld "key" (markiert durch Rot im Bild) sucht dieser Endpunkt nach einem Ort welcher, diesen bestimmten PK hat.
``` json title:"Beispiel Ausgabe"
[
  {
    "PK_Ort": 3,
    "Stadt": "München",
    "PLZ": "80331",
    "Straße": "Platz",
    "Hausnummer": "12"
  }
]
```
