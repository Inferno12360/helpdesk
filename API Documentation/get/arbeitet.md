#### bymkey
![[arbeitet_01.png]]
`getarbeitetbmkey`
Durch das Feld "key" (markiert durch Rot im Bild) sucht dieser Endpunkt nach allen Dienstleistungen welche in einem Ticket, mit diesen bestimmten PK hat.

```json title:"Beispiel Ausgabe"
[
  {
    "Mitarbeiter": 1,
    "Kunde": 7
  }
]
```
