#### bykey
`geteigene_firmabykey`
![[Eigene_Firma_01.png]]
Durch das Feld "key" (markiert durch Rot im Bild) sucht dieser Endpunkt nach einer eigenen Firma welcher, diesen bestimmten PK hat.
``` json title:"Beispiel Ausgabe"
[
  {
    "PK_EigeneFirma": 3,
    "Firmenename": "EduSoft"
  }
]
```
