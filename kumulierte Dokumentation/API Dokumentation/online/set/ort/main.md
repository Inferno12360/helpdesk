Method: `setortmain`
Der Endpoint `setortmain` kann in `ort` erstellen.
Dieser Endpunkt muss mit folgenden Parametern aufgerufen werden:
`Stadt` mit Type `string`
`PLZ` mit Type `string`
`Strasse` mit Type `string`
`Hausnummer` mit Type `string`
